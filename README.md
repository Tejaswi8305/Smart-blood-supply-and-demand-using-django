# 🩸 Smart Blood Supply & Demand Forecasting System

## 📌 Overview

The Smart Blood Supply & Demand Forecasting System is a Django-based web application that uses Machine Learning algorithms to predict future blood demand and improve blood bank management. The system helps reduce shortages, optimize supply, and support hospitals with timely availability of blood units.

---

## 🎯 Objectives

* Predict future blood demand using ML models
* Improve efficiency in blood bank management
* Reduce blood shortages and wastage
* Provide a centralized platform for donors, hospitals, and admins

---

## ⚙️ Features

* 🔮 Blood demand prediction using Machine Learning
* 🏥 Hospital blood request management
* 🧑‍🤝‍🧑 Donor registration and tracking
* ⚠️ Shortage alerts and notifications
* 📊 Data visualization (charts/graphs)
* 🔐 Admin dashboard for system control

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** Django (Python)
* **Machine Learning:** Scikit-learn, Pandas, NumPy
* **Database:** SQLite
* **Visualization:** Matplotlib / Seaborn
